  • POST
    /users/register

    Register

    This method allows a new user to register. See the required parameters section below for an explanation of the variables that are needed to register a new user.

    Request URL:

    http://ticaret.herokuapp.com/services/app/users/register

    Parameters

    • first name Required, String, this is the first name of the user
    • last name Required, String, this is the last name of the user
    • email Required, String, this is the main id helping user to be identified
    • password Required, String
    • location Optional, String
    {
    "userEmail": "mail",
    "userPassword": "pass",
    "userFname": "Ali10",
    "userLname": "ok",
    "userLocation": "tunis"
}

{
  "operation": "register",
  "message": "failed",
  "flagId": null,
  "id": 0,

  "text": "First name is not valid.
    Last name is not valid.
    Email is not valid
    Password must be between 8 & 20 characters.
    "
}

  • GET
    users/[user_id]/events/[event_id]/state

    Check invitation state

    This method allows you to check invitation state : Going, Maybe, or Decline and by as default state None.

    Request URL:

    http://ticaret.herokuapp.com/services/app/users/[user_id]/events/[event_id]/state

    Parameters

    • user_id Required, Integer, this is the id of your user.
    • event_id Required, Numeric, this is the id of the event you would like know user's state.

    Sample Response

    {
    "iuId": 7,
  "iuDate": 1386474347034,
  "iuState": "go"
}
  • GET
    users/[user_id]/events/[event_id]/state/[state_code]

    Change state

    This method allows you to change the state of an event invitation from default state to one of these states : Going, Maybe or Decline.

    Request URL:

    http://ticaret.herokuapp.com/services/app/users/[user_id]/events/[event_id]/state/[state_code]

    Parameters

    • user_id Required, Numeric, this is the id of your user
    • event_id Required, Numeric, this is the id of your event
    • state_code Required, String, this is the state code : Going, Maybe or Decline

    Sample Response

    {
  "operation": "change-state",
  "message": "ok",
  "flagId": null,
  "id": 0,
  "flagText": null,
  "text": null
}
  • GET
    users/[user1_id]/invite/[user2_id]/to/[event_id]

    Invite

    This method allows you to invite a user to a given event.

    Request URL:

    http://ticaret.herokuapp.com/services/app/users/[user1_id]/invite/[user2_id]/to/[event_id]

    Parameters

    • user1_id Required, Numeric, this is the id of your user, the invitor
    • user2_id Required, Numeric, this is the id of your user, the invited
    • event_id Required, Numeric, this is the id of your event

    Sample Response

  "operation": "invite",
  "message": "invited",
  "flagId": null,
  "id": 0,
  "flagText": null,
  "text": null
